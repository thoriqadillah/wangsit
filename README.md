# WANGSIT KBMSI

## Prerequisite
- Node js v16.16.0
- PHP v8
## Collab
```
clone https://github.com/thoriqadillah/antrian-rs-api.git
cd wangsit
composer install
php artisan key:generate
npm i

```
## Run Local Server
Untuk run Laravel:
```
php artisan serve
```
Untuk run tailwind css
```
npm run dev 
```
